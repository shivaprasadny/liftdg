# App Store listing draft

- Title: LiftDG
- Subtitle: Track Every Rep
- Promotional text: Offline strength and cardio tracking with plans, advanced sets, progress, and user-controlled backups.
- Keywords: workout,fitness,strength,cardio,gym,sets,reps,offline,training

LiftDG is an offline workout tracker for strength training, cardio, reusable plans, advanced set structures, history, and personal progress. Build plans, log sets/reps/weight, record cardio, use supersets/drop sets/circuits, review history and records, and export or restore local data. No account is required.

Privacy summary: workout data stays in the app's local device storage; users control exports. See the draft privacy policy.

Screenshot plan: Home; Exercise Library; Plan; Active Workout; Advanced Group; History; Progress; Cardio; Settings/Backup. Screenshots must be captured from the real app and must not be fabricated.
