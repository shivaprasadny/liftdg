## Troubleshooting

- App will not open in Expo Go: install the Expo Go version compatible with SDK 54, or use a development build.
- Active workout: open Start and choose Resume. Set edits persist to SQLite immediately.
- Backup: Settings → Data Management → Create full JSON backup. Store it somewhere you control.
- Restore: choose Restore JSON backup, inspect the summary, then select Merge or Replace. LiftDG creates a local pre-restore snapshot.
- Lost data: without an exported or operating-system backup, locally deleted/uninstalled data may be unrecoverable.
- Reset: export first, type `RESET`, and confirm. Built-ins remain; user data is removed.
